    *Action:* Add `defer` or load `checkout.js` asynchronously to prevent blocking the HTML parser during initial page rendering.

7.  **Add `aria-label` & Labels to Inputs and Buttons (Accessibility Impact: High)**

8.  **Inject H1 Headings on Core Pages (SEO Impact: Medium)**
