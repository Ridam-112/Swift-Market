# AI SEO (GEO / LLM Optimization) Audit

This audit evaluates **swiftmart.space** for discoverability, indexability, and answerability by generative search systems (Google AI Overviews, Perplexity) and Large Language Models (ChatGPT, Gemini, Claude).

**Overall AI SEO Score: 15/100**  
*(The score is extremely low because the site is a client-side rendered Single Page Application (SPA). Crawlers that do not execute heavy client-side JavaScript will only scrape a blank `<div id="root"></div>` and a splash screen, rendering the entire product catalog, shop lists, and internal pages invisible to AI engines).*

---

### Category-Wise AI SEO Scores

| Category | Score | Status | Recommendation |
| :--- | :---: | :---: | :--- |
| **Content & Answerability** | 20/100 | ❌ | AI crawlers find zero body text or structured Q&A formats on raw page fetches. | Implement Server-Side Rendering (SSR) or Static Site Generation (SSG) to output static text. |
| **Entity SEO** | 15/100 | ❌ | Search engines cannot link entities (SwiftMart -> Balurghat -> Local Shops -> Products) without schema or textual relationships. | Deploy JSON-LD entity nodes (`LocalBusiness` and `Product` schemas). |

| **AI Files** | 20/100 | ⚠️ | `robots.txt` does not block AI agents, but the site lacks an `llms.txt` file and HTML sitemap. | Create an `llms.txt` file in the root directory. |
| **Schema Integration** | 0/100 | ❌ | No structured JSON-LD data exists in the code to feed AI Knowledge Graphs. | Add comprehensive `Product`, `LocalBusiness`, and `FAQ` schemas. |
| **AI Overview Readiness** | 10/100 | ❌ | The lack of static content means LLM scrapers cannot quote or summarize this site. | Serve pre-rendered HTML content to search bots. |

### Top 20 AI SEO Improvements

1.  **Implement Static Rendering (SSR/SSG):** Pre-render the React application so crawlers scrape fully formed HTML instead of a blank container.
2.  **Create an `llms.txt` File:** Place a markdown-formatted file in the root directory summarizing your service, locations, and catalog structure specifically for LLM parsers.
3.  **Add a LocalBusiness JSON-LD Schema:** Explicitly link the entity "SwiftMart" to the region "Balurghat" for local AI searches.
4.  **Inject Structured Product Schemas:** Include prices, ratings, and availability in schema format on all product pages.
5.  **Build a FAQ Section with FAQPage Schema:** Provide answer-first Q&As about delivery zones and times to feed Google AI Overview callouts.
6.  **Create an HTML Sitemap:** Host a scannable index page of all shops and categories to help LLM crawlers map the database.
7.  **Optimize the `robots.txt` for AI Agents:** Add explicit rules allowing user-agents like `GPTBot`, `ClaudeBot`, and `PerplexityBot`.
8.  **Structure Content with "Answer-First" Formatting:** Write short, direct answers to common queries (e.g. *"We deliver groceries in Balurghat in 10 minutes"*).
9.  **Implement Breadcrumb Schema:** Let AI crawlers understand category relationships (e.g. `Home > Products > Groceries`).
10. **Include Freshness Indicators:** Add publication and updates dates in schema/metadata to signal the catalog is active.
11. **Include Citations and Sources:** Reference local partners or licensing authorities (like FSSAI) to establish authority.
12. **Optimize Heading Outlines (H1/H2):** Create a clear, nested heading outline so AI engines can parse document outlines easily.
13. **Add Entity-Rich Product Descriptions:** Expand descriptions to mention raw ingredients, packaging, and specific brands.
14. **Avoid Non-Standard Jargon:** Use standard terms (e.g., "delivery fee", "operating hours") so AI models can correctly map properties.
15. **Use Descriptive Alt Text on Images:** Help multimodal models (like GPT-4o or Gemini 1.5 Pro) read and contextualize product images.
16. **Publish localized "Guides" (Content Clusters):** Add a blog section about local food, shopping guides, and merchant highlights in Balurghat.
17. **Minimize Render-Blocking Third-Party JS:** Ensure the page loads fast enough for crawler timeout limits (under 3 seconds).
18. **Add Social Profile Mentions (SameAs Schema):** Link your business profiles (Facebook, Instagram, LinkedIn) inside the Organization schema.
19. **Deploy Table-Formatted Data:** Format product specifications (nutrition, weight, brand details) using semantic HTML `<table>` tags.
20. **Enable XML Sitemap Discovery:** Place the sitemap path directly in `robots.txt` so AI indexers find all pages instantly.

---

### Missing GEO (Generative Engine Optimization) Opportunities

*   **Location-Specific Conversational Q&A:** AI searchers query engines conversationally: *"Who delivers medicine in Balurghat after 9 PM?"* If the page text does not contain these exact Q&A patterns, SwiftMart won't rank in conversational results.
*   **Knowledge Graph Associations:** Lacking an Organization schema means Google's Knowledge Graph cannot verify that "SwiftMart" is a legitimate delivery service in Balurghat.

3.  **Inject Organization & LocalBusiness Schemas:** Deploy the JSON-LD schemas onto the homepage.
